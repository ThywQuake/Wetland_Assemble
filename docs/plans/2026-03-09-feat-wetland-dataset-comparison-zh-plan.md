# 湿地数据集质量对比分析计划

## 概述
本计划旨在建立一套系统的方法，用于评估和对比多个全球湿地卫星数据集的质量。我们将根据数据集的特征进行分类，并应用定制化的对比方法，充分利用高分辨率参考数据（如 GWD30）以及代码库中现有的分析工具。

## 数据集分类

根据 `docs/datasets/` 中的信息，数据集分为以下几类：

| 类别 | 数据集 | 分辨率 | 类型 | 核心用途 |
| :--- | :--- | :--- | :--- | :--- |
| **高分辨率参考** | GWD30, Sentinel-2 | 30m | 动态/分类 | 局部验证/基准 |
| **中分辨率静态** | G2017, GLWD v2 | 232m - 500m | 静态/分类 | 热带/全球基线 |
| **粗分辨率动态(分数)**| GIEMS-MC, SWAMPS, TOPMODEL, WAD2M | 0.25° (~25km) | 动态/淹没百分比 | 全球水文模拟 |
| **粗分辨率动态(掩膜)**| Berkeley-RWAWC (CYGNSS) | 0.01° (~1km) | 动态/掩膜 | 热带水文动力学 |

## 对比分析策略

### 1. 数据协调与预处理
为了实现直接对比，必须对数据集进行一致化处理：
- **空间对齐**：将所有数据集重采样到统一网格：
  - 30m：用于局部高分辨率验证。
  - 0.01°：用于热带地区区域分析。
  - 0.25°：用于全球趋势分析。
- **分类体系映射**：将不同的分类方案（如 GLWD 的 33 类 vs GWD30 的 14 类）映射到统一的**共识湿地类别**（如：有林湿地、无林湿地、开阔水体）。
- **二值化转换**：为所有产品生成“湿地 vs 非湿地”掩膜，以便进行总体精度评估。

### 2. 具体对比方法

#### A. 分类精度评估（对比高分辨率参考）
**数据集**：GWD30, G2017, GLWD v2。
**方法**：
- **混淆矩阵**：计算总体精度 (OA)、生产者精度 (PA)、用户精度 (UA) 和 Kappa 系数。
- **空间精度**：调用 `src/wetland_analysis/analysis/accuracy.py` 中的 `calculate_spatial_accuracy`。
- **特定类别分析**：评估特定类型（如 G2017 vs GWD30 中的红树林）的识别准确率。

#### B. 动态淹没百分比精度（对比聚合后的高分辨率数据）
**数据集**：GIEMS-MC, SWAMPS, WAD2M, TOPMODEL。
**方法**：
- **空间聚合**：将 30m 的 GWD30 数据聚合到 0.25° 的百分比分值。
- **统计相关性**：计算 Pearson 和 Spearman 相关系数。
- **误差分析**：计算 RMSE 和偏差 (Bias)（使用 `src/wetland_analysis/analysis/comparison.py` 中的 `compare_datasets`）。

#### C. 时间趋势一致性
**数据集**：所有动态产品（GWD30, GIEMS-MC, Berkeley-RWAWC 等）。
**方法**：
- **Mann-Kendall 检验**：检测湿地面积是否存在显著变化趋势。
- **Sen's Slope**：对比随时间变化的斜率（变化幅度）。
- **趋势同步性**：评估各数据集是否捕捉到相同的季节性峰值和年际异常变化。

#### D. 不确定性与一致性制图
- **共识制图**：识别多个数据集达成一致或存在显著差异的区域。
- **熵分析**：利用 `src/wetland_analysis/analysis/uncertainty.py` 中的 `calculate_entropy` 映射数据矛盾的“热点区域”。

## 参考数据获取 (Sentinel-2)

由于本地缺少高分辨率参考数据，必须有针对性地获取 Sentinel-2 影像，作为验证的“地面真值”。

### 1. 数据规格要求
- **产品等级**: **Sentinel-2 Level-2A (L2A)** (底层反射率/大气校正后)。
- **分辨率**: 10m (可见光/近红外) 与 20m (短波红外)。
- **核心波段**: 
  - **10m**: B2 (蓝), B3 (绿), B4 (红), B8 (近红外)，用于 NDVI 计算与可视化。
  - **20m**: B11, B12 (短波红外)，用于水体/水分探测 (mNDWI)。
- **采样策略**: 
  - **雨季峰值**: 捕捉最大淹没范围。
  - **旱季枯水期**: 区分永久性湿地与季节性湿地。

### 2. 时空范围
- **时间范围**: **2018 - 2022** (与 GWD30、Berkeley-RWAWC 和 GIEMS-MC 的最佳重叠期)。
- **优先样区 (Reference Sites)**:
  - **亚马逊盆地**: 热带雨林与大型河漫滩。
  - **刚果盆地**: 热带泥炭地与森林湿地。
  - **东南亚红树林**: 滨海湿地验证 (对比 G2017/GWD30)。
  - **潘塔纳尔湿地**: 验证动态趋势一致性的关键区域。

## 实施阶段

### 第一阶段：数据获取与协调
- **Sentinel-2 提取**: 使用 Google Earth Engine (GEE) 或 Sentinel Hub 导出优先样区的中值合成影像（干/湿两季）。
- **体系映射**: 编写 GLWD v2 与 GWD30 分类体系的对照转换脚本。
- **分幅系统**: 设置基于 MGRS 的分幅系统，用于高分辨率对比（参考 `src/wetland_analysis/utils/mgrs_tiling.py`）。
- **标准化**: 将获取的 Sentinel-2 数据重采样至 10m/30m 网格，以便与 GWD30 直接对比。

### 第二阶段：静态基线对比
- 对比泛热带地区的 GLWD v2 和 G2017 数据。
- 利用 GWD30 的 10 年平均值验证静态产品。
- 生成空间一致性分布图。

### 第三阶段：动态与趋势分析
- 对粗分辨率分数产品进行逐月相关性分析。
- 在热带地区对比 Berkeley-RWAWC (CYGNSS) 与 GIEMS-MC 及 GWD30。
- 执行 2013-2022 重叠时段的 Mann-Kendall 趋势分析。

### 第四阶段：报告生成与可视化
- 利用 `src/wetland_analysis/visualization/reports.py` 自动生成 Markdown 评估报告。
- 根据验证结果制作“分区域最佳数据集”推荐图。

## 验收标准

- [ ] 为所有分类产品计算了空间精度指标 (OA, Kappa)。
- [ ] 为所有动态产品生成了时间相关性图谱。
- [ ] 完成了 2013-2022 时段的趋势一致性 (Sen's Slope) 评估。
